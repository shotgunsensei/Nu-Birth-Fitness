YOUTUBE DATA FLOW
Use the YouTube Data API in this order:

STEP 1
Resolve the channel using:
- channels.list
- part=snippet,contentDetails,brandingSettings,statistics
- forHandle=@NU-BirthFitness

STEP 2
From the channel response, get:
- channel id
- title
- description
- thumbnails
- subscriber count if public
- uploads playlist id from contentDetails.relatedPlaylists.uploads

STEP 3
Fetch all uploaded videos using:
- playlistItems.list
- part=snippet,contentDetails
- playlistId=<uploads playlist id>
- maxResults=50
- paginate through all pages until complete

STEP 4
Collect all video IDs and enrich them using:
- videos.list
- part=snippet,contentDetails,statistics,player,status
- batch requests in chunks
- only include videos that are embeddable and public where applicable

STEP 5
Fetch channel playlists using:
- playlists.list
- part=snippet,contentDetails
- channelId=<resolved channel id>
- paginate through all pages

STEP 6
For each playlist the user opens, fetch items with:
- playlistItems.list
- part=snippet,contentDetails
- playlistId=<playlist id>
